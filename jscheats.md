**Data Types**

- Numbers           :var age = 23
- Variables         :var X
- Text (strings)    :var a = "init"
- Operations        :var b = 1 + 2 + 3
- Boolean           :var c = true
- Constant numbers  :const PI = 3.14
- Objects           : var name = {firstN: "John", 
-                                lastN:"Doe}
